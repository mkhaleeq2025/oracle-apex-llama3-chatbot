# Ollama Setup on OCI Ubuntu VM

This directory contains scripts and instructions to install and run Ollama with the LLaMA 3 model on an Oracle Cloud Infrastructure (OCI) Ubuntu compute instance.

## Environment
- Oracle Cloud Infrastructure (OCI)
- Ubuntu Linux (20.04 / 22.04)
- Ollama
- LLaMA 3

## Steps Overview
1. Install Ollama on Ubuntu
2. Pull the LLaMA 3 model
3. Start Ollama service
4. Test Ollama REST API

These scripts are used to support an Oracle APEX chatbot application that communicates with Ollama via REST API.
